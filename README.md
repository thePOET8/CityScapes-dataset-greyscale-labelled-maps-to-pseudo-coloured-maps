# CityScapes-dataset-greyscale-labelled-maps-to-pseudo-coloured-maps
CityScapes数据集灰度标签图转为伪彩色图


目前搜集到的代码都或多或少存在个别问题，针对CityScapes数据集在训练网络时可能需要将灰度标签图转为伪彩色图这一需求，

其中的三个程序分别针对不同需求：

**灰度标签图转为伪彩色图(main.py）**         
---------------------------------------------------------                              
 目前可搜集到的方法，但存在局限性。
